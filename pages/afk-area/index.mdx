# Overview

This documentation is for version `v1.1`

AFK Area is a Spigot Server Plugin to reward your players for being AFK on the server in a certain area.
I have highly kept it configurable and has many features packed in it. The plugin is performance friendly :)

## How to install?

1. Stop your server if it's running
2. Download the plugin JAR from which platform you purchased our plugin.
3. Copy the plugin jar into your server's `/plugins/` folder
4. Start your server. Plugin is installed successfully

## Supported Versions
The current plugin version `v1.1` supports minecraft servers from 1.8 upto 1.20.1
