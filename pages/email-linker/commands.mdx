
## Player Commands and Administrator Commands


## Player Commands

| Command             | Description                                        | Permission                       | Usage                                |
|---------------------|----------------------------------------------------|----------------------------------|--------------------------------------|
| `/email help`       | Basic Help Command for the plugin                  | `No Permission Required`         | `/email help`                        |
| `/email link`       | Command to link player's email address             | `emaillinker.command.link`       | `/email link <EmailAddress>`         |
| `/email resend`     | Resend the verification email to the player        | `emaillinker.command.resend`     | `/email resend <EmailAddress>`       |
| `/email code`       | Verify the sent code from your email to the server | `emaillinker.command.code`       | `/email code <VerificationCode>`     |
| `/email disconnect` | Disconnect your email from the server's database   | `emaillinker.command.disconnect` | `/email disconnect <ConnectedEmail>` |

## Administrator Commands

| Command         | Description                            | Permission                   | Usage                |
|-----------------|----------------------------------------|------------------------------|----------------------|
| `/email reload` | Reloads the configuration files        | `emaillinker.command.reload` | `/email reload`      |
