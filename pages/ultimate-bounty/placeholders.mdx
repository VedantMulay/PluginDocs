# Placeholders

The plugin currently only has 2 placeholders for the Placeholder API. More will be added in the future updates with more features.
| Placeholder | Description  |
| - | :- |
|`%ultimatebounty_bounty_total%`|Get the total amount of money for all the active bounties on the player|
|`%ultimatebounty_total_<Player>%`| Get the total amount of money for all the active bounties of a player|
