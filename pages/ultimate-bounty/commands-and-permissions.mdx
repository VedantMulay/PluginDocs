# Commands & Permissions

## Player Commands

* `/bounty <player> <amount>` - Places a bounty on a player - `No Permission Required`

* `/bounties <player>` - View the active bounties of a player - `No Permission Required`

## Admin Commands
* `/ubounty remove <id>` - Remove a active bounty of a player - `bounty.admin.remove`
* `/ubounty removeall <player>` - Remove all active bounties of a player - `bounty.admin.removeall`
* `/ubounty info <id>` - View info about a active or expired bounty - `bounty.admin.info`
* `/ubounty help` - Get plugin help message - `bounty.admin.help`
