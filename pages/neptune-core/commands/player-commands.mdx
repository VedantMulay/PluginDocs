## Player Commands



| Command | Info | Permission |
| --- | --- | --- |
| `/lobby` | Sends the player back to lobby servers | neptune.command.lobby |
| `/onlinestaff` | Shows the usernames of online staff on the server | neptune.command.onlinestaff |
| `/message <player> <message>` | Sends a message a online player on the network | neptune.command.message |
| `/reply <message>` | Sends a message to the last person messaged | neptune.command.reply |
| `/neptunecore` | Prints the info about the plugin | NONE |
