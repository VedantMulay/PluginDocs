## Commands

| Command | Info | Permission |
|--- | --- | --- |
| `/friend add <player>` | Send a friend request to a player | `neptune.command.friend` | 
| `/friend accept <player>` | Accept a friend request from a player | `neptune.command.friend` | 
| `/friend deny <player>` | Deny a friend request from a player | `neptune.command.friend` | 
| `/fmsg` | Message a online friend in the server | `neptune.command.friend` | 
| `/friend remove <player>` | Remove a player from friend list | `neptune.command.friend` |
