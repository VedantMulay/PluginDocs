# Plugin Commands

The plugin itself does not have that many commands.

| Command                   | Description                                                                     | Permission                 |
|---------------------------|---------------------------------------------------------------------------------|----------------------------|
| `/delivery [delivery-id]` | Used to open the delivery GUI menu. The `delivery-id` parameter is not required | No Permission              |
| `/delivery-master`        | Main plugin command shows help menu if use has permission                       | `delivery.commands.admin`  |
| `/delivery-master reload` | Reloads the plugin configurations. **(Server restart is recommended)**          | `delivery.commands.reload` |
